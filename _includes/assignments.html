starmos.io

  • Inspired by flat design, starmos.io is an online web game created using node.js. The objective of the game is to explore the map from a selection of spaceships, and find asteroids, comets, or enemy players in order to unlock new abilities. All graphics were designed using simple shapes and colors, some of which are generated during gameplay.

  • Technologies Used:

    Vue 3
    pixi.js
    Socket.io
    Express
    Webpack
    GSAP
    Kubernetes
    Docker
  • Created November 2019 – February 2021

ARBnb

  • ARBnb is an augmented reality application developed for Android, and built for LEAN Techniques. Designed in Unity using Google's ARCore, ARBnb provides an interface for rental properties. Interactive tooltips and icons are augmented into the rental via the phone's camera, displaying information to the user on relevant items in the property, such as public hiking trails or room info.

    Landlords add objects to a scene by tapping anywhere in the augmented environment, and select the icons and text to display. Each room's state persists across multiple sessions, allowing tenants to view the scene.

  • Technologies Used:

    C#
    Unity
    Google ARCore
    Google Cloud Anchors
    GCP
    Android SDK
  • Team: created September 2020 – November 2020

Data Exploration Interface

  • The IPMP Data Exploration Interface is an online webapp built using ArcGIS for javascript and Web AppBuilder Source. The webapp displays pavement data for jurisdictions across Iowa, and provides standard features of GIS applications, such as an attribute table, data selection, managing layers, and toggling basemaps. Other features include infographics for specific fields in the pavement data, and cost estimates for potential road treatments. Featurelayers from 2013 through 2020 are available to view and download as a geodatabase or shapefile.

  • Technologies Used:

    node.js
    Arcgis API
    ESRI Plugins
    dojo
    REST API
  • Created September 2019 – March 2020

Lab Assistant

  • Lab Assistant: a companion app for teacher assistants or professors to use in the classroom. Designed for Android, instructors can track student attendance, assign homework, organize students into groups, and manage other details of their classroom or lab. Students can use the app to chat with other users about class material, and notify the instructor for help. The app connects with a springboot backend, accessible via a REST API – which connects to a MySQL database that stores all student, chat, and classroom information.

  • Technologies Used:

    Java
    Gradle
    MySQL
    Springboot
    Maven
    Docker
  • Team: created February 2019 – May 2019

LEGv8 ISA Emulator

  • A subset of ARMv8's Instruction Set Architecture, LEGv8 is a reduced form of assembly with standard commands such as 'ADD', 'LSL', and 'LDUR'. Assembly is passed into this LEGv8 Emulator, which parses and executes each statement. Memory is explicity allocated into static arrays for registers (X9, X10, X11...) and variables (e.g. X0 after executing the above 'LDUR' statement). The emulator is written completely in C, and was created as a class project for COM S 321: Introduction to Computer Architecture and Machine-Level Programming.

  • Technologies Used:

    C
  • Team: created September 2019 – November 2019

Roguelike Game

  • A roguelike game written in C & C++ that runs inside the terminal, with graphics managed by ncurses . Maps are randomly generated: hallways are marked with '#', rooms as '.', enemies as alpha characters, and items as special characters. The player is controlled via keyboard input, and must navigate through the dungeon to find enemy characters and collect items. The game ends once either the player or all enemies are defeated.

  • Technologies Used:

    C
    C++
  • Created February 2019 – May 2019

Infinity Maze

  • This was an early hobby project developed for Android. The game draws an infinite horizontal maze, the path generated using depth-first search. The player controls a pawn that must continually solve the maze as it scrolls faster and faster. All game logic was built with java, using the libGDX api for graphics and sprite objects.

  • Technologies Used:

    Java
    Android SDK
    libGDX
    Gradle
  • Created September 2017 – January 2018
